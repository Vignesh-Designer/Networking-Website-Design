# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/4d3eed50-9067-4b31-aac3-1de4c4ff939e

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

Key Features:
Immersive 3D Scene: A central "Core" sphere with distortion effects, surrounded by a rotating ring and floating data nodes. The entire scene rotates dynamically as you scroll from top to bottom.

Technical Stack:

Framer Motion: For complex scroll-linked animations and layout transitions.
Tailwind CSS: For the "crafted" UI styling and responsive design.
Lucide React: For consistent, futuristic iconography.
